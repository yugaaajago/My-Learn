# VLAN Configuration

Pada materi ini saya menggunakan dua MikroTik CHR. CHR-A untuk trunk dan CHR-B untuk access ke VLAN 10 dan VLAN 20. Tujuan materi ini adalah untuk menguji cara pembuatan VLAN dan testing VLAN itu sendiri, berhasil atau tidak. Disini saya mengetesnya dengan cara mematikan trunk atau VLAN pada CHR-A, alhasil pada CHR-B tidak dapat ping VLAN 10 maupun VLAN 20.
